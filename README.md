# 💰 Student Expense Tracker

A simple and powerful Python-based expense tracker designed for students to manage their daily expenses efficiently.

---

## 🚀 Features

* ➕ Add expenses (name, amount, category, date)
* 📋 View all expenses
* 💰 Total spending calculation
* 📊 Category-wise analysis
* 🗓️ Filter expenses by month
* 🗑️ Delete expenses
* 💾 Data saved using JSON (persistent storage)
* 📈 Graph visualization using Matplotlib

---

## 🛠️ Tech Stack

* Python
* JSON (for storage)
* Matplotlib (for graphs)

---

## ▶️ How to Run

1. Clone the repository or download the code
2. Install required library:

```bash
pip install matplotlib
```

3. Run the program:

```bash
python expense_tracker.py
```
